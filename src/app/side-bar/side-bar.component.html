Using the map hover over a stream.

The stream will be highlighted in blue. Upstream streams will be highlighted in green and downstream in red.

As you zoom in more detailed streams will become visible.

Clicking a stream will display the stream information in the side panel and change the selected streams to be shown using thicker lines.

NOTE: double clicking to zoom will clear the selected stream, mouse wheel zooming will retain the selected stream.

 Search
  1. Click the search tab.
  2. Enter the search term to show the list of matches.
  3. Select the match from the list
  4. Click the search icon in the search field to zoom the map to the items location
